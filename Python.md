# **READ THE README FILE IN EACH ONE BEFORE YOU USE THEM**
 - All of these scripts use **[PYTHON](https://www.python.org/downloads/)** and you need Python installed to PATH - **[PYTHON PATH INSTALLATION GUIDE](https://www.youtube.com/watch?v=H7iqfv5dzRY)**
 - A lot of these scripts use **[FFMPEG](https://ffmpeg.org/download.html)** and you need Ffmpeg installed to PATH - **[FFMPEG PATH INSTALLATION GUIDE](https://www.youtube.com/watch?v=JR36oH35Fgg)**
 - You can use **[VS CODE](https://code.visualstudio.com)** to edit the scripts easier, some of these scirpts will need you to edit them a little bit
 - If you are unable to Extract the scripts, you need to download **[WINRAR](https://www.win-rar.com/start.html?&L=0)**
 - **ALL SCRIPTS HAVE A "START" FILE USED TO LAUNCH THEM**
 - **ALL SCRIPTS HAVE A README EXPLAINING HOW TO USE THEM**

# Media Tools
 - [Simple Video Editor](https://drive.google.com/file/d/14frXvXBWlG7RNhFu2NRHyYig90yeuueQ/view?usp=sharing): free fully modular video editor
 - [Adaptive Scaler](https://drive.google.com/file/d/1useU2Ycy4FEfOj2rWIBy_owtMuo7WjET/view?usp=sharing): Scale videos to the size of your choice (resolution, not quality)
 - [Adaptive Scaler Plus](https://drive.google.com/file/d/1REiG7OKYRnjcD752FGPlLPwyl9kv6K1M/view?usp=sharing): Same as Adaptive Scaler, but DOES enhance quality with resolution
 - [File Size Shrinker](https://drive.google.com/file/d/11oTTGS6-khixbGXcUVPVYNiww7WG5-9y/view?usp=sharing): Shrink videos without messing up quality (file size, not resolution)
 - [Frame Alignment Fixer](https://drive.google.com/file/d/1b-ZQtvFvSeIioH7XNvnJp-rUTVryLa5j/view?usp=sharing): Used to fix frameskips from exported videos
 - [ReEncoder](https://drive.google.com/file/d/1YC92GHE4bUCbjG0VYmOF-IErBGu2F1vf/view?usp=sharing): Used to ReEncode your Videos, without having to edit or ReRecord them
